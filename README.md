# pzc 的工具箱

#### 介绍

pzc 的工具箱，包含：

1. babel 插件以及执行器
2. jscodemod 插件以及执行器
3. posthtml 插件以及执行器
4. postcss 插件以及执行器
5. swigger转ts声明文件以及请求模板生成
6. 项目注释提取和文档生成
7. 文件夹信息统计和分类
8. 文件内容正则匹配、替换和去重
9. 文本格式化
10. excel与json互相转化
11. 文件批量重命名和复制粘贴移动

未完待续...
